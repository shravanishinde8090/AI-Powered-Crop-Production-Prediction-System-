# 🌱 Crop Production Prediction System

This is a Machine Learning project that predicts crop production based on various agricultural parameters.

## 🚀 Features
- Predict crop yield using ML model
- Interactive UI using Streamlit
- Data visualization with Plotly

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Accuracy
Model Accuracy: 94.9%

## ▶️ Run Project
streamlit run app.py

⚠️ Note: Model file is not uploaded due to size limits.
You can retrain the model using the dataset.
